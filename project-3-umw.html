Case Study

UMW Pump Monitoring

An IoT-enabled predictive maintenance system capturing real-time vibration and temperature data, integrated with a cross-platform mobile app and automated ticketing.

Key Tech

IoT & Predictive Maintenance

Tech Stack

  • Quasar Framework
  • Vue.js
  • IoT Sensors
  • Android / iOS
  • Real-time Data

Context

VISI Sdn Bhd

Client: UMW Lubetech

01. The Problem

UMW Lubetech required a way to monitor the health of critical industrial pumps remotely. Manual inspections were inefficient and often failed to catch early signs of mechanical stress such as overheating or excessive vibration - leading to unexpected downtime.

Furthermore, when issues were identified, there was no centralized workflow to track the maintenance process from detection to resolution.

02. The Solution

We developed a comprehensive IoT Web-Based Pump Monitoring System. Hardware sensors were installed to capture real-time telemetry, which is instantly visualized on a central web dashboard.

To ensure accessibility for mobile engineers, we utilized the Quasar Framework to build a single codebase that deployed native-like applications for both Android and iOS, allowing staff to monitor pump health from anywhere on the factory floor.

03. My Contribution

  • Cross-Platform Development: Led the development of the mobile application using the Quasar Framework, ensuring seamless performance and UI consistency across both Android and iOS devices.
  • IoT Integration: Implemented the data pipeline to capture and visualize critical metrics, specifically vibration, temperature, and power consumption.
  • Workflow Automation: Developed a custom E-Ticketing System integrated directly into the platform. This module allows operators to create, assign, and track maintenance tickets whenever an issue is detected.
  • Alerting Logic: Programmed a threshold-based notification system. If sensor data exceeds safety limits (e.g., temperature spikes), the system automatically triggers immediate alerts to maintenance teams.

04. Key Features

  • Real-Time Telemetry: Live monitoring of Vibration, Temperature, and Power usage to detect anomalies before failure.
  • Automated E-Ticketing: A digital workflow to monitor issues that happen, ensuring every alert is logged and resolved.
  • Threshold Alerts: Intelligent logic that triggers notifications instantly if data points reach critical thresholds.
  • Hybrid Mobile App: A unified mobile experience for on-the-go monitoring built with Quasar.
