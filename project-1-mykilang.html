Case Study

MyKilang MES

A comprehensive Manufacturing Execution System built to digitalize factory operations through OEE monitoring and asset tracking.

Role

Technical & Project Lead

Architecture, Team Management

Tech Stack

  • Laravel
  • Vue.js
  • Bootstrap
  • MQTT/IoT
  • MySQL
  • Linux Ubuntu
  • Real-time Dashboards
  • Rest API

Context

VISI Sdn Bhd

01. The Problem

Factories required a digital solution to replace manual tracking of production efficiency. They needed real-time visibility into machine downtime, production counts, and asset utilization to make data-driven decisions.

02. The Solution

I led the development of MyKilang, a centralized Manufacturing Execution System (MES). The system integrates with IoT sensors to capture live data, which is processed to calculate Overall Equipment Effectiveness (OEE).

The solution includes modules for production planning, asset tracking, and interactive dashboards that display live performance metrics.

03. My Contribution

  • Leadership: Served as Technical Lead, defining the system architecture and managing the full project lifecycle.
  • Development: Architected the backend using Laravel and built reactive front-end interfaces for real-time data visualization.
  • Integration: Implemented MQTT protocols to handle real-time data streams from factory machinery.
  • Optimization: Mentored the development team to ensure code quality and performance scalability.

04. Key Features

  • OEE Calculation: Automated calculation of Availability, Performance, and Quality metrics.
  • Asset Tracking: Digital logging of machine location and status.
  • Real-time Monitoring: Live dashboards providing instant feedback to factory floor managers.
